# RushExpress Super Monorepo

This repository contains the RushExpress delivery platform mono-repo with backend services, web applications, and mobile apps.

## Structure
- `backend/`: Django REST API + Channels + Celery.
- `web-customer/`: Customer-facing web app (Vite + React + Tailwind).
- `web-rider/`: Rider-facing web app.
- `web-merchant/`: Merchant-facing web app.
- `web-admin/`: Admin web app.
- `mobile-customer/`: Flutter customer app.
- `mobile-rider/`: Flutter rider app.
- `mobile-merchant/`: Flutter merchant app.

## Status
Initial monorepo structure created. Implementation work for backend, web, and mobile apps will follow.
